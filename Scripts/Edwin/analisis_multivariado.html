Etapa 3 — Evaluacion inicial de relaciones multivariadas (Seccion 5.3.8)

Dataset \(D_{\mathrm{mod}}\): data_agrupada_mod.csv (cuadrante × semana, 2024–2025).

Complementa el analisis bivariado de la Seccion 5.3.7 evaluando redundancia simultanea entre covariables climaticas (VIF, PCA, agrupamiento jerarquico), asociacion parcial clima–respuesta y una regresion exploratoria de diagnostico. Objetivo: recomendar un subconjunto de predictoras y definir la variable respuesta para la Etapa 4.

Metodologia aplicada:

  1. VIF del bloque climatico (multicolinealidad).
  2. PCA exploratorio del bloque climatico (scree + loadings).
  3. Dendrograma de variables climaticas por distancia de correlacion.
  4. Correlacion parcial precipitacion–n_eventos controlando temperatura y viento.
  5. Regresion exploratoria (diagnostico, no modelo final).
  6. PCA del bloque operativo para dimensiones de impacto (frecuencia vs. duracion).

calcular_vif <- function(data, variables, response = "n_eventos") {
  vars_ok <- variables[variables %in% names(data)]
  fmla <- as.formula(
    paste(response, "~", paste(vars_ok, collapse = " + "))
  )
  modelo <- lm(fmla, data = data)
  vif_vals <- car::vif(modelo)
  if (is.matrix(vif_vals)) {
    vif_vals <- vif_vals[, 1]
  }
  tibble(
    variable = names(vif_vals),
    vif = round(as.numeric(vif_vals), 2)
  ) %>%
    arrange(desc(vif))
}

4. Correlacion parcial y regresion exploratoria

Correlacion parcial de Spearman entre precip_total_semana y n_eventos, controlando temp_media_semana y viento_medio_semana (residuos de rangos).

partial_spearman <- function(data, x, y, controls) {
  rank_data <- data %>%
    select(all_of(c(x, y, controls))) %>%
    drop_na() %>%
    mutate(across(everything(), rank))

  resid_x <- lm(as.formula(paste(x, "~", paste(controls, collapse = " + "))), data = rank_data)$residuals
  resid_y <- lm(as.formula(paste(y, "~", paste(controls, collapse = " + "))), data = rank_data)$residuals
  cor(resid_x, resid_y, method = "pearson")
}
